Help

Importing Decklist

To get started, you need to import your deck from a deck builder, like TappedOut or Archidekt. The import list must be in the following format:
1 Card Name
3 Card Name
Click the Import button to load the specified cards.


Choosing a Commander

Once the cards have loaded, you will need to pick a commander for your deck. The first section of the card viewer will show a list of legal commanders from the imported cards as long as no commander is selected. To choose a commander, click the security icon below the card. To deselect a commander and move the card back to the main deck, click the person_remove icon. The app supports adding multiple commanders, but does not check for the legality of your deck.


Choosing Card Art

Clicking the style icon below a card will load a list of all printings of that card. Swapping between the printings will update the image in real-time. Additionally, you may choose the image quality of your card art by selecting an option from the Quality dropdown above the card viewer.

Tip: After clicking the style icon, use the up and down arrow keys to quickly switch card arts.


Tokens

When loading the cards from the deck list, the app will also load any associated tokens that are created by cards in the deck list. You can change the art of these tokens just like a normal card.

Tip: Each section of the card viewer is collapsible. To see the tokens without scrolling too far, collapse the Main Deck section.


Card Backs

To use a custom card back when exporting, insert a link to an image url in the Card Back Image URL field, above the card viewer. A preview of the image will be displayed in the Card Back section of the card viewer if a valid image is provided. The default MTG card back will be used if no url is given.


Exporting a Deck

Once you are satisfied with your deck, give it a name and click the Export button to download a .json file containing the deck object format for Tabletop Simulator. To import the deck in TTS, find the downloaded file and move it to your TTS Saved Objects folder:
Documents > My games > Tabletop Simulator > Saves > Saved Objects
In Tabletop Simulator, the deck object should now show up in the Objects > Saved Objects menu. Drag the object onto your table, and your cards should appear. There will be multiple piles:

  • Main Deck
  • Commander(s)
  • Tokens
  • Double-Sided Cards

Troubleshooting

If you run into any issues while using the app, you can help me identify the errors. If a card fails to import, or you notice other weird behaviors within the app, open up the dev tools console by pressing F12. Check the console tab for any red error messages and send me a screenshot.

If you have an issue with your cards loading in TTS, send me your deck list so that I can investigate the issue locally.
